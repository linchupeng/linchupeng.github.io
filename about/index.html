关于我

94年人，被外行贴了「程序员」标签，其实只是一枚烂大街的码仔，主要方向是服务端开发。目前还没远大的抱负，也没什么成就，只想好好赚钱，提高生活水平。 很开心能在这行工作，现在是一枚后端渣男，曾经努力想做一枚 Androider，奈何没人要，怒转 PHP。现在也写写 Go 和 Python 喜欢 Coding ，水平很菜，但却不甘平凡，喜欢编程带来的挑战和成长。

喜欢简洁的东西，有点强迫症，可能会为一张壁纸满世界找好几天，但也可能一张壁纸就可以用好几年。喜欢 Android ，IOS，其实是喜欢手机，超喜欢手机开箱撕膜那瞬间。

常逛 WP7吧，知乎，头条，最近喜欢打王者（很遗憾没有上王者）(2020年现在已经是嘴强王者)，对事情有自己看法，喜欢自己去折腾东西，但往往有拖延症，简单闷骚，喜欢跟有想法的人在一起，不喜欢没有上进心的人。
